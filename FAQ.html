What files can I upload?



-> You can upload files containing text image with the extension of JPEG,JPG and PNG.
->Camera photos from gallery.
->You can even enter url of image.


How do I view/receive result?



After the text is successfully recognized, links to the newly created output files appear in the user's documents. You can either open the file for viewing or download it to your PC.




What is the maximum file size limit I can upload?



The maximum file size limit is currently 5 MB. To Upload Image larger than 5 MB, please contact our customer service.


How does the choice of recognition languages influence the result?



If there are words and sentences in different languages in the document, e.g., in English and in Hindi, in order to achieve the most accurate result it is recommended to set up the required languages, when recognizing the text.
